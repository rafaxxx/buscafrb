BuscaFRB

Projeto de classificação automática de rajadas rápidas de rádio (Fast Radio Bursts – FRBs) utilizando técnicas de Machine Learning e Deep Learning aplicadas a dados provenientes de radiotelescópios.

O objetivo é desenvolver e avaliar modelos capazes de distinguir FRBs reais de eventos não-FRB, utilizando dados observacionais armazenados em arquivos FITS.

O código principal está implementado em um Jupyter Notebook, contendo todo o pipeline de processamento, desde a leitura dos dados até o treinamento e avaliação dos modelos.

**Requisitos**
Antes de executar o projeto, instale as seguintes ferramentas.

1. Python
Recomendado: Python 3.9+

2. Gerenciador de ambiente (opcional, mas recomendado)
Você pode utilizar:
* Anaconda
* Miniconda
* venv

**Bibliotecas utilizadas no projeto:**
* numpy 
* pandas 
* matplotlib
* seaborn
* scikit-learn
* astropy
* tensorflow / keras
* jupyter notebook

**Como Executar**
1. Clone o repositório
>> git clone https://github.com/SEU_USUARIO/buscafrb.git
>> cd buscafrb

2. Inicie o Jupyter Notebook
>> jupyter notebook
ou
>> jupyter lab

Abra o arquivo:
>> buscafrb.ipynb

Execute célula por célula.
